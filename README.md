# AWS S3 Desktop Console

Aplicação Desktop desenvolvida em Python para gerenciamento de arquivos no Amazon S3.

## Funcionalidades

- Upload de arquivos
- Download de arquivos
- Exclusão de arquivos
- Drag and Drop
- Barra de progresso
- Busca de arquivos
- Estatísticas do bucket
- Suporte a Glacier e Standard

## Instalação

```bash
pip install -r requirements.txt
